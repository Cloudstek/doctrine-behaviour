> Library of different entity behaviours (timestampable etc.)

## Requirements

- PHP 8.1+
- Doctrine ORM 2
- intl extension

## Installation

```shell
$ composer require cloudstek/doctrine-behaviour
```

## Running tests

```
$ composer run-script test
```
